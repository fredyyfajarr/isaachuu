### 🔗 **Live Demo:** [**[isaachuu.vercel.app]**]([isaachuu.vercel.app])

Ini adalah repositori untuk kode sumber portofolio pribadi saya yang dibangun untuk menampilkan proyek-proyek, keahlian, dan perjalanan saya sebagai seorang developer. Dibuat dengan teknologi web modern untuk memastikan performa yang cepat dan pengalaman pengguna yang dinamis.

## ✨ Fitur Utama

* **Desain Modern & Responsif**: Tampilan yang bersih dan dapat diakses dengan baik di semua perangkat, dari desktop hingga mobile.
* **Animasi Halus**: Dibangun dengan **Framer Motion** untuk memberikan efek transisi dan scroll yang interaktif.
* **Dibangun dengan Teknologi Terbaru**: Menggunakan **Next.js (App Router)** dan **Tailwind CSS v4** untuk pengembangan yang efisien dan modern.
* **Struktur Berbasis Komponen**: Kode yang terorganisir dengan baik, memisahkan setiap bagian UI menjadi komponen yang dapat digunakan kembali.

## 🛠️ Teknologi yang Digunakan

* **Framework**: Next.js
* **Styling**: Tailwind CSS v4
* **Bahasa**: TypeScript
* **Animasi**: Framer Motion
* **Deployment**: Vercel
* **Ikon**: React Icons

## 🚀 Menjalankan Proyek Secara Lokal

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di komputer lokal Anda.

**1. Clone Repositori Ini**
git clone https://github.com/fredyyfajarr/isaachuu.git
cd isaachuu

**2. Instal Dependensi**
Gunakan npm untuk menginstal semua paket yang dibutuhkan oleh proyek.
npm install

**3. Jalankan Server Pengembangan**
npm run dev


**4. Buka di Browser**
Buka [http://localhost:3000](http://localhost:3000) di browser Anda untuk melihat hasilnya.
